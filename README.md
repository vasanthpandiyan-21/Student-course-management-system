# Student Course Management System (SCMS)

A Java-based web application to manage students and courses efficiently using Spring Boot.

## 🚀 Features
- Student registration and management
- Course creation and management
- Student-course mapping (enrollment)
- View all students and courses
- CRUD operations for both students and courses

## 🛠️ Tech Stack
- Java
- Spring Boot
- Spring MVC
- Spring Data JPA
- Thymeleaf
- MySQL
- Maven

## 📁 Modules
- Student Module
- Course Module
- Enrollment Module

## 🎯 Project Goal
To build a simple and efficient system for managing students and course enrollment in educational institutions.

## ⚙️ How to Run
1. Clone the repository
2. Create MySQL database (scms)
3. Update `application.properties`
4. Run Spring Boot application
5. Open browser: http://localhost:8080
